# 🚀 FinCal Pro – AI Wealth Architect

FinCal Pro is an AI-powered financial planning and wealth forecasting dashboard built using **Python, Streamlit, and Machine Learning**.
It helps users simulate investments, predict future wealth, analyze risk, and make smarter financial decisions.

This project was developed as part of a **FinTech Hackathon Project**.

---

# 📌 Problem Statement

Many individuals struggle with financial planning because they cannot clearly visualize:

* Future investment growth
* Impact of inflation
* Risk in investments
* Financial independence timelines

FinCal Pro solves this by providing an **interactive AI-powered dashboard** that simulates wealth growth and financial scenarios.

---

# 🧠 Key Features

### 📊 SIP Investment Simulator

* Calculate future wealth based on **monthly SIP**
* Supports **step-up SIP strategy**
* Displays invested vs earned returns

### 📈 Wealth Projection

* Interactive graph showing

  * Future wealth
  * Inflation adjusted value

### 🎲 Monte Carlo Simulation

* Simulates **1000+ investment scenarios**
* Shows possible wealth distribution under uncertainty

### 🔥 FIRE Calculator

* Estimates corpus required for **Financial Independence**
* Compares projected wealth vs FIRE target

### 🧠 AI Wealth Predictor

* Uses **Linear Regression Machine Learning model**
* Predicts future wealth based on investment history

### 📊 Portfolio Recommendation

Risk based portfolio allocation:

* High Risk
* Medium Risk
* Low Risk

### 💎 Net Worth Tracker

Tracks assets such as:

* Stocks
* Crypto
* Cash
* Real Estate

### 📈 Live Market Tracker

Fetches real time stock data using **Yahoo Finance API**

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/nihal8055/FinCal.git
```

Navigate into the project:

```bash
cd FinCal-Pro
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
